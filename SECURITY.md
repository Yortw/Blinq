# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Blinq, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: [yort@users.noreply.github.com](mailto:yort@users.noreply.github.com)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 72 hours. Security fixes will be prioritised and released as patch versions.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
